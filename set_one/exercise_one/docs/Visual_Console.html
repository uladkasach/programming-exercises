Class: Visual_Console

Visual_Console(output_holder)

Creates a Virtual Console object

Constructor

new Visual_Console(output_holder)

This module is responsible for creating a simple to use interface for logging to an arbitrary "visual console". In practice, it simply appends text elements with whatever content is wished to be logged to a predetermined DOM element.
Parameters:
Name Type Description
output_holder DOM the DOM element where output should be appended.

Example
const Visual_Console = require("/utils/visual_console");
    const visual_console = new Visual_Console(dom.querySelector("#console_output_holder"));
    visual_console.log("statement to log") // appends "statement to log" to the visual console
