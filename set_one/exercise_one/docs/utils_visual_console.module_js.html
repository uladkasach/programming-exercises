  • module:utils/visual_console.js This module is responsible for creating a simple to use interface for logging to an arbitrary "visual console". In practice, it simply appends text elements with whatever content is wished to be logged to a predetermined DOM element.

Example

const Visual_Console = require("/utils/visual_console");
const visual_console = new Visual_Console(dom.querySelector("#console_output_holder"));
